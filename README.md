## What is this?
This dumps Sober offsets which can't be dumped manually, more offsets and methods can be found [here](https://discord.gg/SV3m4eHJXp)

## How to use?
`g++ main.cpp -o compiledbytreek.elf2025`<br>
`./compiledbytreek.elf2025`
By the way, don't forget to update VisualEngine pointer offset in Offsets.h
